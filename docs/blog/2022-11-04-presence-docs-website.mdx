大概花了一周时间，终于搭建出了 Presence 文档网站。

这个网站会介绍 Presence 的基本概念，以及如何使用 Presence 来构建一个应用。

这里的文档包括 Presence、Presence React、WebTransport Polyfill、Presence Components 等。

Presence 不仅仅是一个库，在未来，它应该是一个生态，它会提供很多实用的工具，这些工具可以帮助你更好的构建实时应用。
