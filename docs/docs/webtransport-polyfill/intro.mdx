<H2>什么是 WebTransport Polyfill？</H2>

[WebTransport](https://www.w3.org/TR/webtransport/) 是一项类似 WebTransport 的全新通讯技术，最大的不同点在于没有队头阻塞，这也是 WebTransport 做实时应用的一个优势。
Presence 底层就是基于 WebTransport 构建的。
